## 📜 Key Legislation

### Privacy Act 1988 (Australia)
- Governs how personal information is collected, stored, and used  
- Requires organisations to protect sensitive data  
- Establishes principles for privacy and data handling

## 🏢 Organisational Policies

Key policies that support cybersecurity:

- **Acceptable Use Policy**  
  Defines how systems and resources should be used  

- **Incident Response Policy**  
  Outlines how security incidents are handled  

- **Data Protection Policy**  
  Ensures proper handling and storage of sensitive data  

These policies help translate legal requirements into practical actions.

## 🔍 Importance in Practice

Legislation and policies play a critical role in:

- Protecting user and organisational data  
- Reducing security risks  
- Ensuring compliance with legal requirements  
- Providing clear procedures for handling incidents  

They form the foundation for secure and responsible system management.
